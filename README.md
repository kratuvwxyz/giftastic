# Game of Gifrones

Get the latest Game of Thrones gifs from this Giphy.com.

### How to play?

- Click navigation buttons to generate your gifs. 
- Click again the same button to get more.
- Each search gets you 24 gifs.
- Once you get a gif, it is in static mode. You have to click it to play.
- You can search for a specific topic and it will generate a button for you.
- Try “You know nothing, Jon Snow” in search bar!

### Prerequisites

* Watch 'Game of Thrones' all seasons.

### Built with

- HTML5
- CSS3
- Bootstrap
- w3.css
- JavaScript
- jQuery
- GIPHY API
- AJAX
- Github for versioning
